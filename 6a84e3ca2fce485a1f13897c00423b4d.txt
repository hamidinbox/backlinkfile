Mengenai Server dan Client merupakan sebuah konsep dari layanan yang saling terhubung.

Server sendiri merupakan sistem komputer, sistem komputer ini memiliki banyak layanan di dalam nya. Contoh seperti Layanan untuk penyimpanan Data, biasanya menggunakan server diantara lain FTP Server, MySQL Server, Samba Server, Proxy Server, dan masih banyak lainnya.

Client merupakan sebutan untuk komputer yang menggunakan layanan dari Server, contoh, jika client ingin menggunakan layanan Transfer File maka, Server yang sudah aktif, akan langsung memberikan layanan yaitu FTP Server.  Selain itu jika kita menggunakan layanan web, seperti hal nya blog. Disitu server menggunakan service dari Apache Server pastinya.

Berikut ini kita akan membahas sekilas mengenai Teknologi Server Pada Linux Debian.

Teknologi Server pada Linux Debian beberapa diantara yang umum yaitu : 

- SSH Server (Remote Server )
- DHCP Server (Dynamic Host Configuration Protocol)
- FTP Server (File Transfer Protocol)
- Samba Server (File Server)

Mari kita bahas sekilas dari jenis-jenis server yang sudah di sebutkan tadi :

1. SSH Server
SSH Server merupakan kepanjangan Secure Shell merupakan protokol jaringan kriptografi, yang digunakan untuk melakukan remote dengan menggunakan login, biasanya SSH Server digunakan untuk mengkontrol perangkat dari jarak jauh.

2. DHCP Server 
DHCP Server merupakan kepanjangan dari Dynamic Host Configuration Protocol, Server ini memiliki fungsi memberikan Konfigurasi Protokol secara otomatis kepada client. Yang DHCP Server berikan ini adalah Konfigurasi IP Address, DNS, Informasi jaringan dan lainya.

3. FTP Server
FTP Server merupakan kepanjangan dari File Transfer Protocol, Server ini sering digunakan untuk melakukan proses perpindahan file ke server client, dan manajemen File.
Biasanya untuk menggunakan layanan FTP Server, kita bisa menggunakan aplikasi FileZilla Client. Dengan cara login sesuai dengan username dan password oleh server.

4. Samba Server
Samba Server ini bersifat perangkat lunak yang open source. Samba Server jika di Windows ini seperti dengan sharing file menggunakan LAN ataupun Nirkabel. Dengan Cara memanggil IP Address. (\\ipaddress).

Samba Server ini menggunakan protokol SMB (Server Message Block) yang artinya samba server ini merupakan aplikasi yang digunakan untuk berbagi sumber daya. Sumber daya disini maksudnya adalah file file yang akan di share oleh Samba Server ini.

Untuk mengakses samba server pada linux bisa menggunakan perintah smb://ipaddress.
