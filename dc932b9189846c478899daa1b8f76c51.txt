Cara Menggunakan Script Termux
Bagi anda yang sedang mencari script termuk untuk kebutuhan Klikbot sebaiknya kamu pahami dulu ya tentang apa itu script termux.


Masalahnya sekarang sudah banyak script termux di internst yang gunanya berbeda - beda keperluanya, baca dulu agar bisa menggunakan script termux biar nanti kalau anda menjalakan tidak akan eror.

Script yang anda peroleh di internet tidak bisa langsung di gunakan karena anda harus menginstal file yang akan mau di jalankan.

Berikut file yang akan digunakan buat script bot telegram :

pkg update pkg upgrade
pkg install python
pip install telethon
pip install requests
pip insrall bs4
pip install rsa
pip install async_generator
pip install colorama
pip install -r requirements
cd /sdcard/file script yang di extark
python main.py +62 nama curency

Perhatikan tandanya dan jangan sampai keliru masalah nya kalau anda kliru atau kurang maka akan eror di termux.
