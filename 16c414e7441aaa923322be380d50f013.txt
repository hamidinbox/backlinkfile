Pengertian Kitchen Set
==============================
Kitchen Set adalah seperangkat furniture yang di gunakan untuk menyimpan berbagai perkakas dapur seperti rak piring, rak sendok, tempat oven, kulkas, dispenser, kompor, exhaust dan sebagainya yang sudah di desain agar penggunaan ruangan dapur lebih efektif, efisien dan lebih modern.

Kitchen set adalah sebuah furniture yang wajib di miliki jika anda ingin mempunyai dapur yang bersih, rapih, dan terlihat modern.

Dengan adanya kitchen set dapur menjadi rapih, tidak berantakan karena anda dapat menyimpan semua perkakas dapur ke dalam lemari atau laci yang sudah di sediakan. Anda juga dapat membersihkan furniture ini dengan mudah.

Dengan seiring berkembangnya zaman para desainer interior saling berlomba-lomba menciptakan ide desain baru. Mereka saling berkompetisi demi memenuhi kebutuhan pasar.

Hingga kini kitchen set mempunyai beberapa model atau tipe, di antaranya adalah:
1. kitchen set tipe linear
2. kitchen set tipe linear 2 sisi
3. kitchen set tipe L
4. kitchen set tipe U
5. kitchen set tipe meja island

Bahan yang di gunakan dalam pembuatan kitchen set juga variatif, berikut adalah beberapa bahan yang biasanya di gunakan dalam pembuatan kitchen set:
1. kayu solid
2. multiplek atau plywood
3. blockboard
4. stainless steel atau alumunium
5. decosheet
6. MDF
7. HPL
8. cat duco

Kitchen set juga terbagi menjadi beberapa bagian, di antaranya sebagai berikut:
1. kitchen set kabinet atas
2. kitchen set kabinet bawah
3. table top
4. back panel
