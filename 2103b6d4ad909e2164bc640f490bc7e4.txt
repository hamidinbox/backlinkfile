## Panduan Pemasangan Mod BUSSID

Setelah mengunduh mod pilihan, ikuti langkah-langkah berikut untuk memasangnya di perangkat Android:

1. Buka aplikasi pengelola file di perangkat
2. Navigasi ke penyimpanan internal
3. Cari dan buka folder "Download"
4. Pindahkan file mod ke lokasi: folder BUSSID > Mods
5. Buka aplikasi Bus Simulator Indonesia
6. Akses menu "Garasi"
7. Gunakan tombol navigasi "<" atau ">" untuk mencari kendaraan baru
8. Pilih kendaraan mod dan tekan tombol "Pakai"
9. Selesai! Kendaraan mod siap digunakan dalam mode "Ngelayap" atau "Narik"
