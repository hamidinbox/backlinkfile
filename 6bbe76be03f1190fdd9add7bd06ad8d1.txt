Berapa Standar Gaji Staff Admin di Indonesia
==============================
Perusahaan mana yang tidal membutuhkan staff admin, dimana tingginya kebutuhan admin di perusahaan, pasti anda memiliki pertanyaan berapa gaji staff admin.

Pekerjaan staff admin sendiri semakin beragam, mulai dari admin online shop, sosial media, purchasing, dan lain sebagainya. Tentu tiap industri dan bidang admin memiliki tugas dan tanggung jawab yang berbeda. Dari tanggung jawab itulah yang menentukan besaran gaji seorang staff admin.

Tampa basa-basi, berikut Standar <a href="https://spiritsevent.com/2022/04/27/berapa-standar-gaji-staff-admin-di-indonesia-info-lengkap-semua-pekerjaan/">Gaji Staff Admin</a> di Indonesia masing-masing bidang industri

<h3>
Gaji admin toko</h3>
Posisi admin toko gajinya berkisar Rp800.000- Rp3.000.000. Besaran gaji tersebut ditentukan oleh seberapa besar toko tersebut.
<h3>Gaji admin purchasing</h3>
Bila kamu ingin menjadi admin purchasing, gaji yang bisa kamu dapatkan ada di angka Rp3.900.000 per bulannya.
<h3>Gaji admin finance</h3>
Staff admin finance di Indonesia rata-rata gaji yang didapatkan ada di kisaran Rp3.800.000
<h3>Gaji admin online shop</h3>
Jika kamu ingin bekerja menjadi admin online shop, kisaran gaji yang bisa kamu dapatkan adalah Rp3.500.000-Rp2.500.000.
<h3>Gaji admin warehouse</h3>
Saat ini, kisaran gaji staff admin warehouse atau gudang berada pada range Rp3.400.000.
<h3>Gaji admin rumah sakit</h3>
Seorang admin rumah sakit bisa mendapatkan gaji di angka Rp3.600.000 hingga Rp7.500.000.
<h3>Gaji admin sosial media</h3>
Rata-rata seorang admin sosial media mendapatkan gaji sebesar Rp5.000.000. Tentu nominal tersebut kembali lagi kepada perusahaan tempat bekerja.
<h3>Gaji admin support</h3>
Untuk posisi admin support, rata-rata gaji adalah Rp3.600.000 per bulan.
<h3>Gaji admin logistik</h3>
Mengutip dari Indeed.com, di Indonesia gaji staff admin logistik adalah sebesar Rp3.600.000 per bulannya.
<h3>Gaji admin HRD</h3>
Staff admin HRD bisa mendapatkan gaji sebesar Rp4.100.000 per bulan.
<h3>Gaji admin tambang</h3>
Industri tambang diketahui memiliki gaji yang tinggi. Untuk seorang admin tambang, kisaran gaji yang bisa didapat adalah Rp3.600.000-Rp8.000.000.

Demikian informasi untuk menjawab pertanyaan Berapa Standar Gaji Staff Admin di Indonesia, semoga bisa menjadi referensi bagi anda yang berminat bekerja di posisi staff admin.
